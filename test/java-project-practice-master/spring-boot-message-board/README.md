# SpringBoot实现留言版
## 简介

使用SpringBoot实现的一个非常简单的留言板

## 使用的技术

    开发工具：Intellij IDEA
    数据库：MySQL
    开发框架：SpringBoot
    包括的技术：Spring Boot、Spring Data Jpa、Mysql、Swagger、bootstrap
    
## 快速开始

创建数据库: messageboard

导入项目后，运行DemoApplication.java

## Swagger接口文档

> 使用下面的url打开
> http://localhost:8080/messageBoard/swagger-ui.html
